# HR Payroll Management System

A **web-based HR Payroll Management System** developed using **ASP.NET MVC** and **SQL/MySQL**.  
This application helps organizations manage employee data, process payroll, track attendance, and generate salary reports efficiently.

## 🌐 Project Repository

🔗 **Source Code**  
https://github.com/your-username/hr-payroll-management

## ✨ Features

- Employee Management
- Payroll Processing
- Attendance Tracking
- Leave Management
- Payslip Generation
- Role-based Login (Admin / HR / Employee)
- Reports and Salary Details

## 🛠️ Technologies Used

- **ASP.NET MVC**
- **C#**
- **HTML5**
- **CSS3**
- **Bootstrap**
- **JavaScript**
- **SQL Server / MySQL**
