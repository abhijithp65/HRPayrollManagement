# HR Payroll Management System

A web-based **HR Payroll Management System** built using **ASP.NET MVC** and **SQL/MySQL** to manage employee records, payroll processing, and HR operations efficiently.

## Features

- Employee Management
- Payroll Processing
- Attendance Tracking
- Leave Management
- Payslip Generation
- Role-Based Login (Admin / HR / Employee)
- Reports and Salary Details

## Tech Stack

**Frontend**
- HTML
- CSS
- Bootstrap
- JavaScript

**Backend**
- ASP.NET MVC (.NET)

**Database**
- SQL Server / MySQL
